# 📍 RoadtoJB (주소 기반 데이터 정제 도구)

`RoadtoJB_juso`는 엑셀 파일 내의 주소 데이터를 행정안전부 도로명주소 API를 활용하여 정제하고, 지번 분리 및 PNU 코드 생성을 자동화해주는 웹 기반 도구입니다.

---

## 🚀 주요 기능

1.  **엑셀 파일 업로드 및 분석**
    *   `.xlsx`, `.xls` 파일의 헤더를 자동으로 읽어와 주소 필드를 선택할 수 있습니다.
2.  **데이터 정제 (Refinement)**
    *   **행정구역 분리**: 시도, 시군구, 읍면동, 리 정보를 하나의 필드로 통합.
    *   **지번 정제**: 산 여부, 본번, 부번을 조합하여 깔끔한 지번 형식 생성.
    *   **기타주소 제거**: 주소 뒤의 상세 건물 정보 등을 제거하여 데이터의 순수도 향상.
3.  **PNU 코드 생성**
    *   법정동 코드와 지번 정보를 조합하여 19자리 고유 필지 번호(PNU)를 자동으로 생성합니다.
4.  **결과 다운로드**
    *   정제된 데이터(`roadAddr`, `jibunAddr`, `adminNm`, `jibun`, `PNU`)를 포함한 새로운 엑셀 파일로 저장할 수 있습니다.

---

## 🛠 기술 스택

*   **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
*   **Libraries**: [SheetJS (XLSX)](https://github.com/SheetJS/sheetjs) - 엑셀 파싱 및 생성
*   **Server**: Windows PowerShell (별도 설치 불필요)
*   **API**: [도로명주소 검색 API (행정안전부)](https://business.juso.go.kr/)

## ⚙️ 설정 방법 (API 키 발급)

이 서비스를 이용하려면 행정안전부 주소기반산업지원서비스에서 API 키(승인키)를 발급받아야 합니다.

1.  [주소기반산업지원서비스](https://business.juso.go.kr/) 접속 및 로그인.
2.  `내 시스템 관리` -> `승인키 신청` 진행.
3.  신청 시 **검색 API**를 선택하고, URL은 `http://localhost:8000`으로 등록합니다.
4.  발급받은 키를 프로젝트 루트의 `config.json` 파일에 입력합니다.

```json
{
  "jusoApiKey": "여기에_발급받은_승인키를_입력하세요"
}
```

## 🖥 실행 방법 (Windows)

이 프로그램은 **파이썬 설치가 필요 없으며**, 윈도우 환경에서 즉시 실행 가능합니다.

1.  `실행.bat` 또는 `run_server.bat`를 더블 클릭합니다.
2.  윈도우 내장 **PowerShell**을 통해 자동으로 로컬 서버가 구동됩니다.
3.  브라우저가 자동으로 열리며 `http://localhost:8000`에 접속됩니다.

## 📖 사용 방법

1.  **파일 선택**: 정제하고자 하는 엑셀 파일을 선택합니다.
2.  **필드 선택**: 엑셀 헤더 중 주소가 들어있는 컬럼을 선택합니다.
3.  **처리 시작**: `변환 시작` 버튼을 누르면 API 호출 및 정제가 시작됩니다 (진행률 확인 가능).
4.  **결과 확인**: 하단 테이블에서 변환된 데이터를 미리 확인합니다.
5.  **내보내기**: `결과 다운로드` 버튼을 눌러 `.xlsx` 파일로 저장합니다.

---

## ⚠️ 주의 사항

*   API 호출 시 지연 시간(기본 0.1s)이 설정되어 있어 대량의 데이터 처리 시 시간이 소요될 수 있습니다.
*   `config.json` 파일은 깃허브에 공유할 때 API 키 유출을 방지하기 위해 `.gitignore`에 등록하는 것을 권장합니다.
